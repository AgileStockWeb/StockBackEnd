# StockBackEnd
## Require
python 3.7
## Quick setup
Install flask and requests
```
pip install flask
pip install requests
```
Run
```
flask run
```
DataBase前置安裝
```
安裝POSTGRE:https://www.enterprisedb.com/downloads/postgres-postgresql-downloads   最上面那個15.3版  安裝時密碼預設0000
PY安裝資料庫操作相關套件: pip install psycopg2
```
